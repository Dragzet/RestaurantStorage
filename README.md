## Стек технологий
- Frontend: HTML, CSS, JavaScript
- Backend: Golang
- База данных: PostgreSQL
- Деплой: Docker
- Веб-сервер: Traefik

## Установка

1) Склонируйте репозиторий
   ```bash
   git clone https://github.com/Dragzet/RestaurantStorage.git
   cd RestaurantStorage
   ```
2) Запустите приложение
    ```bash
   make
    ```
